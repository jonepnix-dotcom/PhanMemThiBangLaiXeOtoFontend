# Driving License Test Frontend

A modern frontend web application for online driving license examination and realtime driving practice communication.
The project simulates an online automobile driving license learning system with realtime video/audio communication support.

---

# 📌 Project Overview

This project was developed to practice and demonstrate frontend development skills, realtime communication, and modern web technologies.

The system allows users to:

* Practice driving theory tests
* Join realtime video/audio calls
* Communicate with instructors
* Track test progress
* Simulate online driving learning experience

### 1. Clone repository

```bash id="m9r6i6"
git clone https://github.com/yourusername/driving-license-frontend.git
```

---

### 2. Install dependencies

```bash id="sm9g3y"
npm install
```

or

```bash id="n0m4q4"
yarn install
```

---

### 3. Configure environment variables

Update the existing `.env` file:

```env id="8d42bp"
VITE_API_URL=http://localhost:5000/api
VITE_SOCKET_URL=http://localhost:5000
```

Example:

